⚡ ExecuteMsg Variants
🏷 Auction Module
ReceiveNft
CW721 hook — routes NFTs to bid/deposit/swap
Wraps ReceiveNftAction: Bid | DepositMega | SwapDeposit. Funds guard: rejects any native tokens.
FinalizeAuction
{ auction_id: u64 }
Callable by anyone after end_time. Transfers Cosmic to winner; moves winner NFTs to pool; sets status → Finalizing.
WithdrawBid
{ auction_id: u64 }
Self-claim: losers withdraw their own escrowed NFTs. Only when Finalizing or Cancelled.
CancelAuction
{ auction_id: u64 } 🔐 admin
Only if zero bidders. Returns Cosmic to original depositor.
🔄 Swap Pool Module
ClaimSwap
{ requested_ids: Vec<String> }
After SwapDeposit, claim N pool tokens for N staged tokens. Strict 1:1 ratio, no overlap, no dupes.
WithdrawStaged
{ }
Cancel a pending swap. Returns all staged tokens to caller.
🔐 Admin Controls
SetPaused
{ paused: bool } 🔐 admin
Freezes/unfreezes all ReceiveNft operations. Withdrawals remain available.
ProposeAdmin
{ new_admin: String } 🔐 admin
Step 1 of two-step admin transfer. Sets pending_admin.
AcceptAdmin
{ } 🔐 pending_admin
Step 2: clears pending_admin, promotes caller to admin.
ForceCompleteAuction
{ auction_id: u64 } 🔐 admin
Unsticks a Finalizing auction when losers refuse to withdraw. Sets status → Completed.
UpdateConfig
7 optional fields 🔐 admin
Patch any config value except admin address (use ProposeAdmin for that). Validates default_min_bid ≥ 1.

All NFT deposits enter via CW721 SendNft → ReceiveNft hook. The inner msg field carries a ReceiveNftAction that routes the call.
